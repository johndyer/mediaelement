MediaElementPlayer.js

1. Single MP4 File

This demo demonstrates how to add volume level display to the mute toggle button. Raise and lower the volume and watch how the button icon changes to reflect the changes in volume. This implementation provides up to 100 points of precision.

The class prefix name associated with the volume level can also be change by setting: ""volumeIconLevelClassPrefix" in ths options.

<video width="640" height="360" src="../media/echo-hereweare.mp4" type="video/mp4" 
	id="player1" poster="../media/echo-hereweare.jpg" 
	controls="controls" preload="none"></video>

$('audio,video').mediaelementplayer({
    volumeIconLevels: 2
});

.mejs-controls .mejs-mute.mejs-volume-level-1 button,
.mejs-controls .mejs-mute.mejs-volume-level-2 button {
    background: url(../build/volume-levels-2.png);
}

.mejs-controls .mejs-mute.mejs-volume-level-1 button {
    background-position: 0px 0px;
}

.mejs-controls .mejs-mute.mejs-volume-level-2 button {
    background-position: 0px -16px;
}

Note: if you are working with local files, you'll need to add your working directory to the Flash Global Security Settings.
